# 拡充ロードマップ（ambient-bureaucracy）

> この本を「どう太らせるか」の計画。2026-07-11 の1on1で確定した原則に沿う。
> 大方針：**エビデンス先行・翻訳フレーム・背骨先行**。書きやすい順ではなく、主張の核から固める。

## 貫く4原則（1on1で確定）

1. **Evidence-first**：章を書く前に、その章を支える一次発言を `evidence/` に採取する。**引用は捏造しない**。一次に届かない主張は本文で断定しない。
2. **Translation-framed**：各章は**特定の懐疑者**（＝「DX＝脱お役所仕事を正義と信じる同僚」）に向けて書く。**逆説を1文で着地させてから、機構・証拠をその"従属"として置く**（パンチ→証拠の順。逆にすると相手が脱落する＝道場#1の学び）。
3. **Backbone-first**：`thesis` → 第4章（新官僚主義）→ 第5章（環境知能）から固め、前提（第2章）と未来（第7章）で挟む。
4. **完了基準は"他者翻訳"**：自分が安心したかではなく、**懐疑者が「たしかに」と半歩動く**か。

## 本書 MVP の完了基準（＝どこまでで"一度手放す"か）

> **懐疑者ペルソナが読んで"お役所仕事に戻す"に半歩うなずく状態**。具体的には：
> - 第2・4・5章のドラフトが揃う（DXの帰結 → 新官僚主義 → 環境知能の一本道）。
> - 各章に**一次エビデンス2本以上**（原文＋出典）が編み込まれている。
> - `thesis.md` の一文テーゼが確定し、環境知能まで橋が架かっている。
