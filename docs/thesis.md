# 背骨（Thesis）— この本が主張する唯一のこと

> 章がぶれたら、ここに戻る。**本の主張は1文に畳める**べきである。

## 一文テーゼ（暫定）

> **生成AI時代の正解は「脱お役所仕事」ではなく、かつて捨てたお役所仕事の構造をAIに肩代わりさせ、人間の認知負荷を消す『新官僚主義』であり、それが現場に届く唯一の形は動線を変えない『環境知能』である。**

## なぜ「逆説」なのか（読み手の予測を裏切る核）

- DX言説の主流＝「お役所仕事＝悪、脱却こそ正義」。
- 本書＝「その脱却が個人の脳をパンクさせた。AI時代はお役所仕事に**戻す**方が効く」。
- この逆張りが刺さるのは、**まだ誰もこの世界観を言語化していない**から。逆に、この世界観を前提共有しないまま実践Tips（上司バックログ等）だけ出しても「時代に逆行」としか読まれない（＝本書が"実践例"の前に"世界観"を建てる理由）。

## この本にしかない価値（差別化）

- 一般論・ハウツーは生成AIのディープリサーチで無価値化した。
- 残る価値＝**「地獄のような現実の中で、テクノロジーをどう現場（手袋）にリーチさせたか」の一次実践**と、**それを世界のAIリーダーの一次発言で理論化した体系**の結合。
- 前者は著者の実務（復帰後の炎上現場・上司バックログ）、後者は `evidence/` の一次資料。

## 論証の3本柱（各章はこのどれかに接続する）

1. **負荷の柱**：脱お役所仕事 → 認知負荷の爆発（ブルシットジョブ再考）。
2. **構造の柱**：生成AIは構造化プロセスに最も強い → お役所仕事の構造は最適な"AIの器"。
3. **リーチの柱**：環境知能＝動線を変えない実装だけが現場に届く（手袋の逆説）。

## 新官僚主義 → 環境知能への「橋」（要の一節）

> 本書で最も落としやすい継ぎ目。道場#1でも新官僚主義止まりで環境知能へ届かなかった。ここを1本の論理で繋ぐ。

1. **新官僚主義＝構造を固める**：生成AIは入力（固めた文脈）の質が出力の質を決める。だから決裁フロー・様式・ルールという"お役所仕事の構造"を、AIの燃料として復権させる。
2. **だが"固める作業"を人間にやらせたら本末転倒**：文脈を整え、様式に流し込み、参照タイミングを合わせる——これを人手でやれば、消したはずの認知負荷がそっくり戻る（DXの逆説の再発）。
3. **ゆえに、固める・参照する・整形するを"裏で"自動化する必要がある＝環境知能**：人間側の物理動線を1ミリも変えず、構造を裏で勝手に満たしておく。介護の手袋を脱がせない。
4. **結論**：新官僚主義（構造の復権）は、環境知能（動線を変えない自動充填）とセットで初めて成立する。**片方だけでは、お役所仕事の苦役か、AIの燃料切れか、どちらかに落ちる。**

*このテーゼは仮説であり、エビデンス採取の過程で反証されたら躊躇なく上書きする。*
