# エビデンス採取プロトコル

> 本書の説得力は、**著者の意見ではなく一次発言の骨格**に宿る。ここは「地の文で使う前の素材置き場」。
> 原則：**クレーム → 引用（原文） → 出典URL → 一次/二次 → 反証の有無**を1セットで記録する。二次情報（まとめ記事・要約）だけの引用は本文に載せない。

## 記録フォーマット（コピーして使う）

```markdown
### [出典者/媒体] 主張を1行で

- **クレーム**：本書のどの主張を支えるか（例：第4章 新官僚主義の逆説）
- **原文引用**：> "……"（改変しない。訳は併記し原文を残す）
- **出典**：URL（一次＝本人の発言/論文/公式。二次＝報道/まとめ）
- **種別**：一次 / 二次
- **日付**：発言・公開日
- **反証・留保**：文脈の限定、反対意見、後の撤回など（無ければ「なし」）
- **使いどころ**：どの段落でどう編むか
```

## 品質ルール

1. **一次を必ず1本**：各主張は最低1つの一次資料（本人の X/論文/公式ブログ/登壇）で支える。
2. **原文を残す**：翻訳だけにしない。用語（terminology）そのものが本書のフックになる。
3. **反証を消さない**：都合の悪い留保も併記する。知的誠実さが本書の生命線。
4. **文脈を切り取らない**：発言の前後・意図を歪めない。
